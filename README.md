# pixelprice
PixelPrice es una app web basada en datos para rastrear precios del mercado de furni de Habbo (legacy, sin NFT). Analiza tendencias históricas, volatilidad, liquidez y comportamiento del mercado, y detecta movimientos inusuales como inflaciones o caídas abruptas. Diseñada para traders, coleccionistas y píxeles curiosos. Interfaz en español.
